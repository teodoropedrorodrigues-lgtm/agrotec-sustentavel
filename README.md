# Agro Forte, Futuro Sustentável

## Descrição

Projeto desenvolvido para conscientizar sobre a importância da produção agrícola sustentável e da preservação ambiental.

## Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript

## Funcionalidades

- Layout responsivo
- Cartões informativos
- Botão interativo sobre Agricultura 4.0
- Quiz educativo
- Design moderno com temática agroambiental

## Objetivo

Mostrar que é possível aumentar a produção agrícola utilizando tecnologias sustentáveis que respeitam o meio ambiente.
